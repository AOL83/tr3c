# Security and Export Control

By default, this repository contains no ITAR or EAR controlled content. If there is any uncertainty, stop and escalate to the program security lead.

Secrets, credentials, and license keys are forbidden in this repository.

Escalation procedure:
1) Flag the content as export-controlled in the PR.
2) Notify the security lead and document control.
3) Do not merge until a written determination is recorded.
